### One reframing I want to deposit

The loop `σ-multiplicativity ≡ coord_mul-associativity` has a clean
group-theoretic shape. `dilation_mul_key_identity` says σ is a group
homomorphism *per-application*. The question of whether σ extends to a
homomorphism at the iteration level (σ_(x·y) = σ_y ∘ σ_x) requires
source-associativity (because ∘ on functions is always associative, so
source needs to match if the homomorphism is to be well-defined).

So `dilation_compose_at_beta` is asking: "does σ extend to a group
homomorphism from (atoms-on-l-under-coord_mul) to (dilation group
under ∘)?" The answer requires source-associativity, which is what
we're trying to prove via the homomorphism property. *This is
structurally the same situation as constructing a group from a binary
operation: associativity must be proven independently of the
homomorphism property to any reference group.*

The three exits A/B/C each provide a way to get associativity
*independently* of the σ-homomorphism path:
- (A) dimensional lift via R: associativity is free in 3D, project down
- (B) observer-supplied witness: associativity supplied as commitment
- (C) fresh Desargues: triangle setup that captures (a·x)·y =
  a·(x·y) directly

The bin-1 grade diagnostic (s144) sharpens (A) further: (A.1)
construction-grade build of dilation_ext_3D etc reproduces the loop
one level up (s148 monodromy at height 5); (A.2) is a fresh
